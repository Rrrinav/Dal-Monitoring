# Dal-Mointoring System

This was our project submission for project showcase at BIS, Standards Day, SKICC Kashmir.

> **We won 3rd position**

It focuses on SDGs 6, 14 & 15.

It uses S3 bucket, SQS and a light Enet Segmentation model to manage images and mark them.

Then shows the heatmap of dal on a interactive dashboard.
